# 🚀 Task Manager – Full Stack Web Application

A modern full-stack Task Management application built using **React, Node.js, Express, and MongoDB**, deployed with **Netlify (frontend)** and **Render (backend)**.

---

## 🌐 Live Demo

👉 Frontend (Vercel):  
  https://task-manager-card-git-main-akshaya-vasireddi-s-projects.vercel.app/

👉 Backend (Render API): 
  https://task-manager-backend-edt0.onrender.com/

---

---

## ✨ Features

- 📝 Create, Read, Update, Delete (CRUD) tasks
- ✔ Mark tasks as completed / pending
- ⚡ Real-time UI updates
- 📱 Fully responsive design (mobile + desktop)
- ☁️ MongoDB Atlas integration
- 🌍 Fully deployed full-stack application

---

## 🛠️ Tech Stack

**Frontend:**
- React (Vite)
- Axios
- Framer Motion
- CSS

**Backend:**
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- CORS

**Deployment:**
- Netlify (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)

## 🚀 Run Locally

git clone https://github.com/Akshaya-Vasireddi/task-manager.git  
cd backend  
npm install  
node server.js  

cd frontend  
npm install  
npm run dev
